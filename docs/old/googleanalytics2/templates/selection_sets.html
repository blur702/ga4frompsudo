About Property Selection Sets

Property Selection Sets allow you to group GA4 properties together for easier reporting. This is particularly useful when you want to generate reports that aggregate data across multiple properties.

Use Cases

  • Brand Grouping: Group properties by brand or business unit.
  • Website Types: Group properties by website type (e.g., e-commerce, blog, corporate).
  • Geographic Regions: Group properties by geographic region.
  • Custom Reports: Create custom groupings for specific reporting needs.

Benefits

  • Simplified Reporting: Quickly select multiple properties without having to check each one individually.
  • Consistent Grouping: Ensure you're using the same property groups across different reports.
  • Time Saving: Save time by reusing property selections instead of recreating them for each report.
