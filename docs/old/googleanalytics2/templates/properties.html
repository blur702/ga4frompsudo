About GA4 Properties

Google Analytics 4 properties are the foundation of your analytics data. Each property represents a website or app that you're tracking with Google Analytics.

Managing Properties

  • Sync Properties: Fetch the latest properties from your Google Analytics account.
  • Global Reports: Control whether a property is included in global reports that aggregate data across multiple properties.
  • Update Metadata: Fetch the latest metrics and dimensions definitions for a property.

Property Information

  • Property ID: The unique identifier for the GA4 property.
  • Website URL: The URL of the website associated with the property.
  • Last Synced: The last time the property information was updated.
